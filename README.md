# P5js Sudoku Generator

### Print out your own Sudoku puzzle.

#### Instructions

1. Go to the [Sudoku Generator](https://digitalight.github.io/p5js-sudoku/)
2. Select from Easy, Normal or Hard difficulty and then hit the Print button.
